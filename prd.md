# Product Requirements Document (PRD): AI 상세페이지 제작 앱

## 1. 프로젝트 개요 (Project Overview)
본 프로젝트는 사용자가 제공한 기본 정보(제품 이미지, 색상, 폰트 등) 또는 간단한 키워드를 바탕으로 전문적인 모바일용 상세페이지를 생성하는 서비스를 구축하는 것을 목표로 합니다. 특히 이미지나 문구가 부족한 경우 AI를 통해 자체 생성하여 완성도 높은 결과물을 제공합니다.

## 2. 핵심 목표 (Core Objectives)
- **고품질 상세페이지 자동 생성**: 경쟁사 분석 및 검증된 판매 구조를 바탕으로 상세페이지 제작.
- **AI 기반 자산 생성**: 이미지 및 후킹 문구가 없을 시 AI(Gemini)를 통해 생성.
- **모바일 최적화**: 모바일 환경에서 가장 효과적인 레이아웃 및 디자인 적용.
- **사용자 커스터마이징**: AI 생성 후에도 색상, 폰트, 내용 등을 자유롭게 수정 가능.

## 3. 주요 기능 (Key Features)

### 3.1. 상세페이지 생성 엔진
- **브랜드 가이드 반영**: 사용자 지정 색상 코드 및 폰트를 적용한 디자인 생성.
- **이미지 활용**: 사용자가 첨부한 이미지를 레이아웃에 최적화하여 배치.
- **경쟁사 분석**: 검색 기능을 통해 경쟁사 상세페이지를 분석하고, 이를 설계에 반영.
- **구조적 설계**: 일반적으로 판매율이 높다고 알려진 상세페이지 구조(문제 제기 -> 해결책 제시 -> 신뢰 구축 -> CTA)를 기본 템플릿으로 활용.

### 3.2. AI 역할 분담
- **Gemini 3.0 Flash (Visual & Layout Specialist)**
    - 경쟁사 페이지 시각적 분석.
    - 제품 이미지 레이아웃 최적 배치.
    - 전체적인 UI/UX 디자인 생성.
- **Gemini Nano/Banana (Copywriting & Text Analyst)**
    - 구매 유도 문구(카피라이팅) 생성.
    - 사용자 리뷰 분석을 통한 강점 도출.
    - 부족한 텍스트 정보 보충.

### 3.3. 편집 및 공유
- **실시간 수정**: 생성된 상세페이지의 텍스트, 이미지, 스타일 실시간 편집 기능.
- **모바일 앱 형태**: Nextjs 기반의 PWA 또는 모바일 웹 최적화 UI.

## 4. 기술 스택 (Technical Stack)
- **Frontend**: Next.js (App Router 권장)
- **Styling**: Tailwind CSS (유지보수 및 생산성)
- **AI Models**: 
    - Google Gemini 3.0 Flash (Multimodal API)
    - Google Gemini Nano (On-device or specialized text task)
- **Deployment**: Vercel (Next.js 최적화)

## 5. 상세 기능 요구사항 (Functional Requirements)
1. **입력 단계**: 제품 이름, 주요 특징, 색상/폰트 정보, 참고 URL 또는 이미지 업로드.
2. **분석 단계**: 경쟁사 URL 입력 시 AI가 해당 페이지의 강점 및 디자인 요소 추출.
3. **생성 단계**: 
    - 텍스트가 부족하면 AI가 후킹 문구 생성.
    - 이미지가 부족하면 AI 이미지 생성(DALL-E 또는 Stable Diffusion 등 연동 고려 가능, 기본은 Gemini 시각 요소 활용).
4. **결과물**: 섹션별(헤더, 특장점, 상세 정보, 리뷰, CTA)로 구성된 상세페이지.
5. **수정 모드**: 각 섹션 클릭 시 내용 수정, 배경색 변경, 이미지 교체 가능.

## 6. 비기능 요구사항 (Non-Functional Requirements)
- **디자인 품질**: AI가 만든 느낌을 최소화하고, 전문 디자이너가 작업한 듯한 세련된 Look & Feel 유지.
- **응답 속도**: AI 생성 프로세스 중 사용자에게 진행 상황을 시각적으로 표시(Skeleton UI 등).
- **반응형**: 모바일 기기에서의 사용성 최우선.

## 7. 향후 로드맵 (Roadmap)
- Phase 1: 핵심 생성 엔진 및 Next.js 기본 구조 구축.
- Phase 2: 경쟁사 분석 기능 및 AI 문구 생성 고도화.
- Phase 3: 실시간 편집 도구 인터페이스 강화.
- Phase 4: 이미지 생성 AI(필요 시) 추가 연동 및 완성도 개선.
